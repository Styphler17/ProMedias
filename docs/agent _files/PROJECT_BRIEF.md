# Project Brief — Promedias eCommerce Website

## What is being built
A local eCommerce website for Promedias Liège that combines online product sales with repair-service lead generation.

## Core business positioning
- Local electronics store
- Mobile phone/computer repair shop
- Accessories seller
- Refurbished and used device seller
- Facebook-first business needing a stronger owned web presence

## Primary customer journeys
1. Buy a product online and choose delivery or pickup.
2. Find a repair service and contact/book quickly.
3. Verify trust: address, phone, reviews, hours, real store.
4. Browse refurbished devices with warranty and condition info.

## Conversion priorities
- Call now
- Message on WhatsApp
- Visit store
- Add to cart
- Request repair quote
- Reserve refurbished device

## Suggested page modules
### Home
Hero, trust bar, featured categories, top services, featured products, why choose us, FAQ, map/contact footer

### Shop page
Filters, sorting, stock badges, category intro copy, local pickup reminder

### Product page
Gallery, condition badge, specs, warranty, stock, CTA, delivery/pickup info, related products, trust details

### Repair page
Service list, pricing-from, turnaround, supported brands/devices, CTA to call/message/visit

### Contact page
Map, address, hours, phone, Facebook, contact form, directions, FAQ

## Functional features
- Search
- Filters
- Cart/checkout
- Coupon-ready architecture
- Inventory-friendly admin structure
- Review/testimonial section
- FAQ accordion
- Structured data
- Analytics hooks

## Design cues
- Practical retail UI
- Clean cards
- Sharp trust-first hierarchy
- Accent color used sparingly
- Strong mobile usability
