# Content Requirements

## Business facts to confirm from docs
- Official business name formatting
- Logo files and brand colors
- Product categories actually sold
- Whether repairs can be booked online
- Delivery zones and pickup rules
- Payment methods accepted
- Warranty rules
- Returns policy
- Languages required
- Real opening hours

## Minimum homepage copy blocks
- One-line value proposition
- Short paragraph about products + repairs
- 4 service/category highlights
- Trust proof section
- FAQ items
- Contact block with NAP

## Product content minimums
- Clear title
- Price
- Condition
- Stock status
- At least 3 specs where relevant
- Delivery/pickup info
- Warranty note
- Short SEO meta description

## Repair content minimums
- Supported device type
- Problem examples
- Starting price or quote note
- Estimated turnaround
- CTA method
